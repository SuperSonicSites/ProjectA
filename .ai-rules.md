# AI Coding Rules & Architecture

## 1. Hugo Layout Hierarchy
- **Primary Source of Truth**: All layouts MUST reside in `themes/visual-sanctuary-theme/layouts/`.
- **Shadowing Restriction**: DO NOT create or modify files in the project root `layouts/` directory. 
- **Rationale**: Root layouts shadow theme layouts. To avoid fragmented logic and "split-brain" bugs, we maintain the theme as the single source of truth.
- **Documentations**: All implementation summaries should be added to /.docs

## 2. Tech Stack Consistency
- **DSL**: Hugo Templates + Go Logic.
- **Styling**: Tailwind CSS (Utility-first). No inline styles unless absolutely necessary for dynamic AI logic.
- **Backend Tasks**: TypeScript scripts in `/scripts`.
