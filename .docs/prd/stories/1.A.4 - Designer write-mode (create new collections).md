Epic: `1 - PaperPause Autonomy System`

## Goal
Create missing category/collection scaffolding so new schedule entries (e.g., `fantasy/*`) can generate successfully without manual work.

## Scope
- Create (if missing):
  - `content/<category>/<collection>/`
  - `content/<category>/<collection>/_index.md`
  - prompt config (repo-standard location/schema)

## Invariants
- Non-destructive by default:
  - create missing files
  - updates limited to explicit fields only
- Idempotent:
  - reruns do not churn or rewrite unrelated content
- **Model**: `gemini-3-flash-preview` (Thinking: **MEDIUM**).

## Acceptance criteria
- Running Designer on a new schedule entry creates the required scaffolding.
- Re-running Designer produces no changes unless inputs changed.

## Rollback
- Revert created/edited files via git.
